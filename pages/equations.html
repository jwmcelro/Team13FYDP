Equations

Equations and symbols present a unique struggle for students with access needs. Having accessible equations benefits students by ensuring that screen readers can properly read them and makes math searchable in documents.

  • Microsoft Powerpoint
  • Microsoft Word

Microsoft PowerPoint

This workflow includes the following task: best practices, using the math editor, alt text for pictures of math.

1. Best Practices

Following best practice advice for equations helps increase the clarity and readability of material.

Step 1: When creating equations in Microsoft Office, it's recommended to leave it in the original .docx or .ppt format as opposed to saving as a PDF.

Step 2: Insert one equation per line instead of creating one big equation with multiple lines.

Step 3: Try to insert symbols using the equation editor instead of copying one from elsewhere. Text copied from other sources may not be fully supported by assistive technologies.

2. Built-In Microsoft Math Editor

Equations can be created using the equation editor in PowerPoint.

Step 1: Click "Insert" tab on navigation bar, click "Symbols" and select "Equation".


Step 2: Navigate to the Equation tab after creating the equation block to type out the equation.


Step 3: Click the "Equation" button on the top left corner to use some built in equations provided by Microsoft.

3. Adding Alt Text for Math

Ensures that the equation(s) contain text descriptions that can be read by a screen reader to help visually impaired students.

Step 1: Click the border of the equation so there is a solid outline, right click the border and select "Edit Alt Text".



Step 2: Type out a written explanation of the equation so that a screen reader can read it aloud to students.

Microsoft Word

This workflow includes the following task: best practices, using the math editor, using MathType.

1. Best Practices

Following best practice advice for equations helps increase the clarity and readability of material.

Step 1: When creating equations in Microsoft Office, it's recommended to leave it in the original .docx or .ppt format as opposed to saving as a PDF.

Step 2: Insert one equation per line instead of creating one big equation with multiple lines.

Step 3: Try to insert symbols using the equation editor instead of copying one from elsewhere. Text copied from other sources may not be fully supported by assistive technologies.

2. Built-In Microsoft Math Editor

Equations can be created using the equation menu, LaTeX, and Unicode Math.

Step 1: Click "Insert" tab on navigation bar, click "Symbols" and select "Equation".


Step 2: After creating the equation block, navigate to the Equation tab to type out the equation.


Step 3: Click the "Equation" button on the top left corner to use some built in equations provided by Microsoft.

3. Using MathType

MathType is a free add-in that can be used for MS Word. It provides tool to create math and offers greater accessibility.

Step 1: To install click on the "Get Add-ins" button.
Note: If you are using a mac operating system the "Add-ins" button can be found under the Insert tab in your menu bar.


Step 2: Search for MathType in the search bar and click the "Add" button. Once installed, MathType should appear as an option in your Insert menu.


Step 4: Select the MathType button and add equations by typing or handwriting them using the pop-up and select "Insert" when finished.
