Checklist

Images

Ensure images are accessible by using high quality images, avoiding images of scanned documents, and adding alt text to images.

  • Microsoft PowerPoint
  • Microsoft Word
  • Google Slides
  • Google Docs

Microsoft PowerPoint

This workflow includes the following tasks: best practices, adding alt text to images, avoid uploading images of scanned documents.

1. Best Practices

Use high quality photos: Ensure photos are clear by using high quality photos from sites like Flickr.

2. Adding alt text to images

Images need alt text descriptions for screen readers to describe the image to visually impaired students.

Step 1: Right click the image and select "Edit Alt Text".



Step 2: Type 1-2 sentences to describe the image and its context. If the picture is decorative, select "Mark as Decorative".

3. Avoid uploading images of scanned documents

Avoid uploading images of scanned documents such as textbook pages, journal articles or other supplementary materials. To the screen readers, this scan looks like an image without alt text and is unable to assist the student.

Step 1: If available, provide a link to an online textbook and reference textbook questions. If not, type out the excerpt into a Word document following accessibility best practices.
