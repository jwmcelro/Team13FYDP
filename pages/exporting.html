Exporting

Properly exporting your content into a PDF ensures that it remains in an accessible format.

  • Microsoft PowerPoint
  • Microsoft Word
  • Google Slides
  • Google Docs

PowerPoint

This workflow includes the following task: accessibility checker, exporting to PDF.

1. Using the Accessibility Checker

Before exporting, use the built-in Accessibility Checker to ensure every accessibility need is met.

Step 1: Select the review tab in the top bar and click "Check Accessibility".



Step 2: Review every warning and click on each warning to view steps on how to fix them.

2. Exporting Powerpoint to PDF

Export PowerPoint slides to a PDF for a smaller file size and ensure that it is accessible for screen readers.

Step 1: Click File in the top menu and select Export in the left hand side menu.



Step 2: Select "Create a PDF/XPS Document" to export the PowerPoint to PDF.



Step 3: Click the Options button, check off "Document structure tags for accessibility", and click on "OK", then "Save".

Google Slides

This workflow includes the following task: exporting to PDF.

1. Exporting Google Slides to PDF

When a PDF is directly exported from Google Slides, it does not contain any tagging for accessibility, meaning that formatting features such as tabular data, lists, headings, labeled images, and so on, will not be reliably conveyed by screen readers as a result. As a result, your Google Slides needs to first be exported into PowerPoint to make the necessary accessible edits before exporting to PDF.

Step 1: Export the Googles Slides to a PowerPoint by selecting File and then download as Microsoft PowerPoint.

Step 2: Open the PowerPoint, make any formatting edits, and follow accessibility best practices on AccessKit. Next, follow the steps for Exporting a PowerPoint to PDF.
