Graphs

Create accessible graphs by providing alt text, multiple means of distinguishing graph information in addition to colour, and text descriptions of graphs.

  • Microsoft PowerPoint
  • Microsoft Word
  • Google Slides
  • Google Docs

Microsoft PowerPoint

This workflow includes the following tasks: colour choice, alt text, text descriptions.

1. Graph colour choice

Colour should not be the only means of distinguishing information in a graph or chart. This would be inaccessible for colour blind students.

Step 1: Under the Design tab, select "Colour" and click "Grayscale". The example below is accessible because different shades can be distinguished.


Step 2: Ensure that the graph continues to convey the same information when in Grayscale. The example below is inaccessible because the graph is the same shade.


Step 3: If inaccessible, change the colours to be different shades of the same colour i.e. blue. This will allow colourblind students to see the difference in shades. Also, adding labels for each part of the graph will aid with interpretation.

2. Add alternative text to graphs

Graphs need alt text descriptions for screen readers to describe the graph to visually impaired students.

Step 1: Right click on the graph and select "Edit Alt Text".



Step 2: Type 1-2 sentences to describe the graph and close the window to save.

3. Text descriptions for graphs

For complex graphs, alt text may not provide enough information for students using screen readers. Provide a text summary and graph legend near the graph for better context.

Step 1: Provide context of the graph above. Include a graph legend and a description of the data shown in the graph.


Step 2: Provide a text description for the graph. This could be a summary of the trends and/or interpretations.
